# Einstein-Field-Equations
This will be a concise demonstration of solving the Einstein Field Equations using the Schwarzschild method. Then we will discuss how the terms in the solution imply certain properties of black holes.. .

## Schwarzschild Solution

The Schwarzschild metric solves the Einstein field equations in vacuum for a
spherically symmetric mass $M$. In standard Schwarzschild coordinates
$(t, r, \theta, \phi)$ the line element is given by

$$
ds^2 = -\left(1 - \frac{2GM}{c^2 r}\right)c^2 dt^2
      + \left(1 - \frac{2GM}{c^2 r}\right)^{-1} dr^2
      + r^2 \left(d\theta^2 + \sin^2\theta\, d\phi^2\right).
$$

This metric describes the spacetime geometry outside a non-rotating,
spherically symmetric body of mass $M$.
